# Hybrid-Framework
Hybrid Framework with Data Driven and TestNg for our LinkAirways Project 

"This project is designed to execute smoke tests for the FC application, focusing on the selection of domestic routes for both one-way and round-trip journeys.
The test scenarios include bookings for 1 adult, 1 child, and 1 infant, across various cabin fare types—Promo, Economy, Business, and Business Flexi.
The process also covers bookings with and without add-ons, with payments facilitated through Visa credit card transactions."
